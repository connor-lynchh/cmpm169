Imitate

To start this experiment I looked through the examples that were given in lecture and section. One in particular from section piqued my intrest and I started the experiment using the code from the example as a starting point.

Integrate

I saw from lecutre that Tracery could have been used to compelete this assigment. I had experince with the tool before but I didnt want to go for a generative story but instead a linear one. This story would have a set path and would repeat over and over infinetely.

Innovate

For the innovate step I thought that creating a simple story of emotions and things one does in a day through text art was a unqiue thing. The waterfall effect helps with this idea as time moves on with with no breaks in between. As you move from one set of words to the next you can still see the previous which shows how those feelings may still linger even if time has moved on. There is also the idea of the cycle that repeats over and over as the set of words in this program never change.
