Experiment 3 - Generative Methods

Imitate

I started with an example from the open processing website and then quickly found that example to be to complex for what I wanted to do for this experiment.

I then looked at another example on the website that had a cool distortion effect and that was the example I used as the foundation for the assigment

Integrate

During this experiment I made sure to make use of the noise function that was talked about during lecture and section. This was mainly used for how the color of the animation changes each time the animation runs.

Innovate

Through trial and error of trying to work with pots and then dropping that idea for the distortion effect the result was quite intresting. I continously messed around with the distortion effect and noise to create this really nice wave effect. The animation is also smooth with no lag issues
